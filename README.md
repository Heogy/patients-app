# Patient App

## Description

This is a simple application that allows a user to view a list of patients and their details. The application is built
using :

- React
- Material UI
- React Router
- React Query
- Material react table

## Installation

### Install dependencies

```bash
npm install
```

### Run the json server

```bash
npm run api
```

### Run the application

```bash
npm run start
```

### Run the tests

```bash
npx cypress open
```

### Enjoy
